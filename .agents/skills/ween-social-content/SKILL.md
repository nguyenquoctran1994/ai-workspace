# When to use this skill
Use this skill when creating Vietnamese social media content for WEEN JAPAN, including post concepts, captions, and image prompts.

# Required inputs
- audience focus (parents, teenagers, or both)
- content pillar (sleep, recovery, movement, nutrition, habit, motivation, tracking)
- format (carousel, short post, caption, story, image prompt)
- desired tone or message focus

# Output format
- short content concept
- recommended caption
- suggested image prompt
- key style constraints

# Do
- Follow `WEEN_JAPAN/WEEN_CONTENT_SYSTEM.md` content pillars and tone.
- Keep language calm, premium, supportive, and educational.
- Prioritize trust-building before selling.
- Use positive, aspirational Vietnamese phrasing.

# Don't
- Use hype, fear, fake urgency, or miracle height claims.
- Use bodybuilding, medical, or hospital-style language.
- Copy full brand documents; reference them instead.
