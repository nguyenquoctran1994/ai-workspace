# When to use this skill
Use this skill when validating packaging design, copy, and visual consistency for WEEN JAPAN products.

# Required inputs
- packaging type (box, pouch, bottle, pouch)
- target user (parent trust, teen appeal)
- key brand attributes to verify
- current design elements or copy notes

# Output format
- checklist of packaging alignment items
- recommendations for improvements
- callouts for brand consistency

# Do
- Follow `WEEN_JAPAN/WEEN_DESIGN.md` packaging direction.
- Keep the review focused on premium, minimal, Japanese wellness style.
- Confirm packaging avoids hard supplement/gym/beefy aesthetics.

# Don't
- Use medical or pharmaceutical design language.
- Approve loud, aggressive, or gaming-style packaging.
- Copy entire design docs; summarize the relevant rules.
