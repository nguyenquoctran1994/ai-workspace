# When to use this skill
Use when planning or writing daily social or marketing content for CHILLAB.

# Required inputs
- content theme or topic
- channel or format (post, story, caption)
- audience characteristics
- tone preferences

# Output format
- daily content idea
- caption or copy suggestion
- recommended visual mood
- any key reminders or content rules

# Do
- Keep the output concise and easy to adapt.
- Use a consistent voice appropriate for CHILLAB.
- Reference CHILLAB project docs when available.

# Don't
- Assume WEEN-style rules apply unless CHILLAB documentation says so.
- Produce overly long or unfocused copy.
- Duplicate full project documents.
