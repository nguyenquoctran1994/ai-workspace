# When to use this skill
Use when drafting landing page sections, hero copy, trust messaging, and product presentation for WEEN JAPAN.

# Required inputs
- main goal (lead capture, trust, pack sales, coaching signups)
- target audience (parents, teenagers)
- key offer elements (tracking, pack, coaching, habit system)
- preferred style notes

# Output format
- landing page outline with section names
- suggested copy for each section
- design guidance linked to `WEEN_JAPAN/WEEN_DESIGN.md`
- lead form and CTA recommendation

# Do
- Reference `WEEN_JAPAN/WEEN_MASTER.md` for positioning and offer logic.
- Reference `WEEN_JAPAN/WEEN_DESIGN.md` for visual tone.
- Keep messaging calm, premium, and trustworthy.
- Include safe disclaimers and parent-focused trust elements.

# Don't
- Use miracle height promises or exaggerated claims.
- Use aggressive ecommerce or supplement-first language.
- Duplicate the full brand documents inside the output.
