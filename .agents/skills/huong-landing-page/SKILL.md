# When to use this skill
Use when designing landing page copy, sections, and user flow for HUONG.

# Required inputs
- main conversion goal
- audience profile
- key product or service offer
- desired tone or visual style

# Output format
- landing page section outline
- suggested headline and supporting copy
- UX and messaging notes

# Do
- Keep the landing page focused on clear benefits and trust.
- Use concise, reusable sections.
- Reference HUONG project docs when available.

# Don't
- Use vague, unfocused messaging.
- Overload the page with too many competing offers.
- Copy whole project documents.
