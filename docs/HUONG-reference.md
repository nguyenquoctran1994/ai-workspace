# HUONG Reference

HUONG is an external business/media folder with a smaller AI docs area. It is not part of the `ai-workspace` Git workspace and should not be treated like normal source code.

Real path:

```text
C:\Users\ADMIN\Documents\PROJECTS\HUONG
```

Relative path from `ai-workspace`:

```text
..\HUONG
```

Useful AI docs path:

```text
..\HUONG\HUONG_AI
```

Useful landing page project path:

```text
..\HUONG\HUONG_AI\HUONG_LANDING_PAGE_PROJECT
```

Codex should not move HUONG or deeply scan media/product folders.

## Safe HUONG Usage

- Prefer `HUONG_AI` for AI and project docs.
- Only inspect product/media folders when the user asks.
- Ask before editing external business files.
