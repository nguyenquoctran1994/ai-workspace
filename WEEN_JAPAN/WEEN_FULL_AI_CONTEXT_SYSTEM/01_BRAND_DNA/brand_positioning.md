# WEEN JAPAN BRAND POSITIONING

WEEN JAPAN is a hybrid brand with the main direction:

Primary:
- Wellness growth-support system

Secondary:
- Teen growth coaching ecosystem
- Japanese-inspired biotech lifestyle
- Premium wellness habit tracking

WEEN is NOT:
- a medical clinic
- a miracle height promise
- a gym supplement brand
- a childish teen app

Core idea:
WEEN helps teenagers build disciplined daily habits that support healthy growth:
- sleep
- recovery
- movement
- consistency
- nutrition awareness
- guided tracking

Decision maker:
- Parents

Emotional user:
- Teenagers
