# RECOMMENDED TECH STACK

User level:
- beginner / solo founder

Priority:
- low cost
- stable
- easy to deploy
- AI-agent friendly

Recommended MVP stack:

Frontend:
- React + Vite

Styling:
- Plain CSS first or Tailwind later
- Avoid heavy UI libraries at the beginning

Database:
- Supabase

Hosting:
- Vercel

Forms:
- Supabase table insert
- or Google Form for first validation phase

Automation:
- Make.com or n8n later

AI:
- OpenAI API later, after tracking flow is validated

Why this stack:
- cheap
- simple
- common
- easy for AI agents to understand
- good enough for MVP
