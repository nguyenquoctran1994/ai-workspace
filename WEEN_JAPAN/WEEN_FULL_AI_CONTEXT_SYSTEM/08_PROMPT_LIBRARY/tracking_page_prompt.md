# TRACKING PAGE PROMPT

Build or improve the tracking page.

Goal:
Make parent check-in simple and emotionally reassuring.

Must include:
- height input
- sleep input
- exercise check
- supplement check
- note field
- streak/progress display
- gentle motivation

Style:
- mobile-first
- calm
- premium
- Duolingo-style progress but not childish

Avoid:
- pressure language
- medical language
- complex dashboard
