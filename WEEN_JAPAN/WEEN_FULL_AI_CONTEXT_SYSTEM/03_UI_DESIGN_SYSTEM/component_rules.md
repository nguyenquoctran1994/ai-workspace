# COMPONENT RULES

Reuse components first.

Core components:
- HeroSection
- TrackingCard
- ProgressCard
- HabitCheckinForm
- ParentTrustSection
- PackCard
- StatCard
- CTASection

Component style:
- rounded 24–32px
- soft shadows
- clean hierarchy
- high readability
- mobile-first

Avoid creating too many one-off components.
